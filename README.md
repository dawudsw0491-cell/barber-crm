# Barbershop CRM

CRM sederhana untuk membantu operasional barbershop.

---

## Project Status

Planning Phase

---

## Project Objective

Membangun aplikasi CRM berbasis web yang membantu pengelolaan:

* Customer
* Booking
* Service
* Dashboard Operasional

---

## MVP Features

### Dashboard

* Total Customer
* Total Booking
* Total Revenue

### Customer CRM

* Tambah Customer
* Edit Customer
* Cari Customer
* Riwayat Kunjungan

### Service Management

* Tambah Service
* Edit Service
* List Service

### Booking Management

* Tambah Booking
* Update Status Booking
* List Booking
