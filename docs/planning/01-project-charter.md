# Project Background

Sebagian besar operasional barbershop masih menggunakan pencatatan manual untuk pelanggan, layanan, jadwal booking, dan pendapatan.

Metode tersebut menyebabkan kesulitan dalam:

* Melacak riwayat pelanggan
* Mengelola booking
* Mengetahui pelanggan aktif
* Mengukur performa bisnis
* Menyusun laporan operasional

Diperlukan sebuah sistem CRM sederhana yang mampu membantu digitalisasi proses operasional barbershop.

---

# Business Problem

Permasalahan utama yang ingin diselesaikan:

1. Data pelanggan tidak terpusat.
2. Riwayat kunjungan sulit ditelusuri.
3. Booking masih dilakukan secara manual.
4. Tidak tersedia dashboard operasional.
5. Tidak tersedia laporan sederhana untuk pemilik usaha.

---

# Project Goals

## Primary Goals

* Mengelola data customer.
* Mengelola data layanan.
* Mengelola booking pelanggan.
* Menyediakan dashboard operasional.

## Secondary Goals

* Menyiapkan fondasi sistem loyalitas pelanggan.
* Menyiapkan fondasi online booking.
* Menyiapkan fondasi laporan bisnis.

# Success Criteria

## Business

* Seluruh customer dapat tercatat dalam sistem.
* Booking dapat dikelola secara digital.

## Technical
