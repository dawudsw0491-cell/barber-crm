# 2. Purpose

Dokumen ini mendefinisikan kebutuhan pengguna dalam format User Story untuk mendukung pengembangan Barbershop CRM secara terstruktur dan berorientasi nilai bisnis.

Format:

As a [Actor]

I want [Goal]

So that [Benefit]

I want to see dashboard summary

So that I can monitor business performance quickly.

Acceptance Criteria:

* Total Customer displayed
* Total Service displayed
* Total Booking displayed

High
